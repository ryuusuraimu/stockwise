# stockwise
株式投資の基礎知識を検索できる、React + TypeScript製のFAQナレッジアプリです。
株式投資は、初心者にとって専門用語や制度が多く、必要な情報にたどり着きづらい領域です。  
stockwiseは、投資判断そのものではなく、株式投資に関する基礎知識をFAQ形式で整理し、検索語に応じて関連する情報へたどり着きやすくすることを目的にしています。

## Purpose
このアプリは、Helpfeel開発部インターン応募に向けて、ナレッジ検索・FAQ検索・情報設計への関心を形にするために制作しています。
主な目的は以下です。
- 複雑で誤解されやすい知識領域を、FAQ形式で整理する
- ユーザーが入力した言葉から、必要な情報へたどり着く体験を小さく実装する
- React / TypeScript / JSONデータ管理を使った基本的な開発力を示す
- ナレッジ検索やドキュメント化に対する関心を、制作物として表現する
---
## Features
- 株式投資の基礎知識FAQを検索
- 検索語に応じてFAQ候補を表示
- FAQのタイトル、質問文、回答文を表示
- 簡易スコアリングによる検索結果の並び替え
- 該当するFAQがない場合のメッセージ表示
- 外部APIなしで動作するシンプルな構成

## Search Logic
検索語を空白で分割し、FAQのタイトル・質問文・回答文などに含まれているかを確認します。
一致した項目に応じて簡易的なスコアを付け、スコアの高いFAQを上位に表示します。
高度な自然言語検索やAI検索ではありませんが、FAQ検索の基本的な構造を理解しやすくするため、まずは説明可能でシンプルなロジックにしています。

## Why stockwise?
株式投資は、初心者にとって「何を調べればよいか」が分かりにくい領域です。
たとえば、PER、PBR、配当利回り、成行注文、指値注文、NISA、分散投資など、基本的な言葉でも、初学者にとっては意味や使い方を整理するのが難しい場合があります。
stockwiseでは、そうした知識をFAQとして整理し、ユーザーが自分の言葉で検索しながら学べる体験を目指しています。
---
## Connection to Helpfeel
Helpfeelは、ユーザーが本当に知りたい情報にたどり着ける検索体験や、ナレッジの活用に取り組んでいる企業だと理解しています。
このアプリでは、株式投資という専門用語の多い領域を題材に、情報をFAQとして整理し、検索によって必要な知識にアクセスしやすくする体験を小さく実装しました。
実装規模は小さいですが、ナレッジ検索・情報設計・ドキュメント化への関心を、自分なりに形にした制作物です。
